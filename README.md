## 📖 About this application

This application creates a bridge between your computer and text-based AI models. It runs locally on your machine. You do not need to rely on external web portals to interact with the system. The tool provides a user interface that allows you to manage conversations, save chat history, and integrate external code snippets. You choose how the model behaves by selecting specific roles or character files. This setup gives you full privacy and control over your interactions.

## 🔧 Troubleshooting and maintenance

If you face problems, consult this list of common fixes:

* The server does not start: Ensure no other application uses port 8080. If another program uses this port, the software will show an error. Restart your computer to clear the port.
* The application runs slow: Close other memory-heavy software like web browsers with many tabs or video games.
* Settings do not save: Ensure your user account on Windows has read and write permissions for the folder where you installed the software.
* Updates: If you experience bugs, return to the download link and select a newer version of the installer. Run the installer again to overwrite the old files with the latest version.

## 🛡️ Privacy and local control

Your privacy remains the priority for this project. Since the application runs as a local bridge, your data stays within your machine hardware. The software does not send your personal chat transcripts to external servers unless you specifically choose to connect it to an external API service. You manage all imports and history locally. We recommend that you keep your local folder secure if you handle sensitive information.

## 🧩 Modifying behavior with character files

You can customize how the AI responds by using character files. A character file is a small text instruction set that defines the tone, history, and goals of the AI entity. You can find pre-made character templates within the interface. Select "Import Character" to load a new file. This feature enables deep role-play or specific productivity modes.
